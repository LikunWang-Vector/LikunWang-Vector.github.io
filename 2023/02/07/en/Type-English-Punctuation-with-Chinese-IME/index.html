How to Type English Punctuation While Using Chinese Input Method

Created2023-02-08|Updated2023-02-08|Debug Notes
|Word Count:46|Reading Time:1mins|Post Views:

The solution is simple: Just press CTRL + . (period) to switch between Chinese and English punctuation.

This way, when you type letters, pinyin will still appear, but all punctuation will be in English format. This avoids the need to frequently press Shift to switch between input modes.
