#### 2\. 我要先简单了解一下

HTML的语法是很简单的，容错率也很高，可以说不管怎么样它都能跑起来，但是代码的可读性是尤为重要的，尤其对于网站的维护者而言。如果站点出现问题，杂乱无章的HTML代码可读性差，难以让管理员定位，也难以让我们想要借鉴网页的小白查找数据。

所以我们的目标就是知道HTML的基本布局：

```python # bs4解析基础：HTML语法 # HTML 是用来描述网页的一种语言。 # HTML 指的是超文本标记语言 (Hyper Text Markup Language) # HTML 不是一种编程语言，而是一种标记语言 (markup language) # 标记语言是一套标记标签 (markup tag) # HTML 使用标记标签来描述网页 # 其语法核心就是利用不同的标签对网络上的内容进行标记，从而使网页显示出不同的展示效果 # HTML 标签是由尖括号包围的关键词，比如 <html> # HTML 标签通常是成对出现的，比如 <b> 和 </b> # 标签对中的第一个标签是开始标签，第二个标签是结束标签 # HTML 标签也可能单独出现，叫做单标签，比如<br /> # 开始和结束标签也被称为开放标签和闭合标签 # 如 # <h1> # I love U # </h1> # <h2 align="center">I hate U</h2> # h1:标签 align:属性 center:属性值 # 由此可以知道格式： # <标签 属性="属性值">被标记的内容</标签> # <标签 属性="属性值"/> # <标签 /> # <a href="http://www.bing.com">必应搜索</a> # <img src="xxx.jpg"/> # <br /> ``` 

我们知道了HTML源代码本质上就是各种各样标签围成的，每个标签有自己的含义，里面包含着对应属性的数据。
