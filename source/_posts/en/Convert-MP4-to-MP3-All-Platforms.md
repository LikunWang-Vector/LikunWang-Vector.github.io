If you only want to keep the audio track from an MP4 video, you can convert MP4 to MP3 format. MP3 is a digital audio format supported by almost all devices, players, and editors. Whether you're converting MP4 to MP3 for offline playback or further editing, you can extract the audio track and save it as MP3. Here are 8 practical methods to convert MP4 to MP3 without quality loss.

![MP4 to MP3](/images/posts/如何在Windows-Mac-iPhone-Android-/45712f821b63.jpeg)

## Part 1: Free Online MP4 to MP3 Conversion

Online MP4 to MP3 converters are convenient and free. You can change MP4 to MP3 format to extract audio without third-party programs. However, most online converters have file size limits.

**Steps:**
1. Click "Add Files to Convert" on the website
2. Select MP4 videos to convert
3. Click the Audio icon and set MP3 as output format
4. Adjust audio settings (sample rate, channel, bitrate, encoder)
5. Click Convert

![Online Convert](/images/posts/如何在Windows-Mac-iPhone-Android-/f96d48f4d6cf.jpeg)

## Part 2: Windows Media Player (Windows)

Windows Media Player is pre-installed and can convert MP4 to MP3 for free on Windows 10/8.1/7.

**Steps:**
1. Run Windows Media Player, click File → Open to add MP4
2. Click Organize → Options
3. Select "Rip Music" in Options
4. Choose MP3 from Format list under Rip Settings
5. Click Apply and OK

![Windows Media Player](/images/posts/如何在Windows-Mac-iPhone-Android-/0ebda4250196.jpeg)

## Part 3: iTunes (Mac)

iTunes is pre-installed on all Mac computers.

**Steps:**
1. Update iTunes, select Preferences from Edit menu
2. In General section, click Import Settings
3. Expand "Import Using" list, select MP3 Encoder, click OK
4. Add MP4 video to iTunes library
5. Click File → Convert → Create MP3 Version

![iTunes](/images/posts/如何在Windows-Mac-iPhone-Android-/2e47859b8d98.jpeg)

## Part 4: Media Converter (iPhone)

**Steps:**
1. Download and install Media Converter on iPhone
2. Tap + in top right, select Import Video
3. Select Info icon, choose Extract Audio, set MP3 as output
4. Select Convert

![Media Converter](/images/posts/如何在Windows-Mac-iPhone-Android-/635c0c489489.jpeg)

## Part 5: Video to MP3 Converter (Android)

**Steps:**
1. Open Video to MP3 Converter
2. Browse and select MP4 video
3. Select MP3 in Output Type, adjust audio bitrate
4. Select Convert

![Video to MP3](/images/posts/如何在Windows-Mac-iPhone-Android-/fc02d2119734.jpeg)

## Part 6: VLC Media Player

VLC is a multi-platform free open-source multimedia player and converter.

**Steps:**
1. Run VLC, select Convert/Save from Media dropdown
2. Click Add to add MP4 video, then Convert/Save
3. Find Profile under Settings, select Audio-MP3
4. Click Start

![VLC](/images/posts/如何在Windows-Mac-iPhone-Android-/ed2577c51856.jpeg)

## Part 7: HandBrake

HandBrake is a free open-source video converter for Windows and Mac.

**Steps:**
1. Open HandBrake, click Source → Open File to add MP4
2. In Output Settings, select MP3 from Container list
3. Specify profile settings and filters
4. Click Start

![HandBrake](/images/posts/如何在Windows-Mac-iPhone-Android-/7fbd11726119.jpeg)

## Part 8: FFmpeg

FFmpeg can also be your free MP4 to MP3 converter software.

**Steps:**
1. Download, install and launch FFmpeg
2. Copy and paste: `ffmpeg -i filename.mp4 filename.mp3`
3. Press Enter
4. Conversion complete

![FFmpeg](/images/posts/如何在Windows-Mac-iPhone-Android-/614926626eed.jpeg)

## Summary

You can use the above methods to convert MP4 to MP3 on your phone or computer. Whether you want to batch extract or individually extract MP3 audio from MP4 videos, you can choose from the online and offline video converters mentioned above. In most cases, MP4 to MP3 conversion can be completed within minutes.
