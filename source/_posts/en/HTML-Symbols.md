### HTML Symbol Entities

HTML entity symbols were explained in the previous chapter.

Many mathematical, technical, and currency symbols are not present on a normal keyboard.

To add such symbols to an HTML page, you can use the HTML entity name.

If no entity name exists, you can use the entity number, a decimal or hexadecimal reference.

#### Example

```html
<p>I will display &amp;</p>
<p>I will display &#8364;</p>
<p>I will display &#x20AC;</p>
```

Will display as:

```
I will display &
I will display €
I will display €
```

| ∅ | `&#8709;` | `&empty;` | EMPTY SETS |
| ∇ | `&#8711;` | `&nabla;` | NABLA |
| ∈ | `&#8712;` | `&isin;` | ELEMENT OF |
| ∉ | `&#8713;` | `&notin;` | NOT AN ELEMENT OF |
| ∋ | `&#8715;` | `&ni;` | CONTAINS AS MEMBER |
| ∏ | `&#8719;` | `&prod;` | N-ARY PRODUCT |
| ∑ | `&#8721;` | `&sum;` | N-ARY SUMMATION |
