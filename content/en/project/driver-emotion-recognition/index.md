## Project Overview

Developed an intelligent driver emotion recognition system as my graduation design project, focusing on real-time emotion detection to enhance driving safety through advanced computer vision techniques.

## Key Features

### Custom Dataset Creation
- Designed and built a comprehensive driver emotion dataset using web scraping techniques
- Collected diverse facial expressions under various driving conditions
- Ensured dataset quality and diversity for robust model training

### Modified YOLOv7 Architecture
- Adapted YOLOv7 network structure for emotion detection tasks
- Optimized model architecture for real-time performance
- Achieved high accuracy in emotion classification

### Real-time User Interface
- Developed intuitive UI using PyQt5 framework
- Integrated SQLite database for data logging and management
- Implemented real-time data streaming capabilities
- Created user-friendly visualization of emotion detection results

## Technical Implementation

### Deep Learning Framework
- **Model**: Modified YOLOv7 for emotion detection
- **Training**: Custom dataset with data augmentation
- **Optimization**: Real-time inference optimization

### Software Development
- **Frontend**: PyQt5 for cross-platform GUI
- **Database**: SQLite for local data storage
- **Data Processing**: OpenCV for image preprocessing
- **Real-time Processing**: Optimized pipeline for live video streams

## Key Achievements

- **High Accuracy**: Achieved precise emotion detection across multiple emotional states
- **Real-time Performance**: Optimized for live video processing
- **User Experience**: Intuitive interface with comprehensive logging features
- **Independent Development**: Completed as solo graduation project
